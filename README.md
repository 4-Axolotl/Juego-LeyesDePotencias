# Juego-LeyesDePotencias
Juego para la practica de las leyes de las potencias

En caso de que no puedas ingresar el simbolo "^" lo cual es un problema comun en 
teclados  en los cuales el simbolo esta detras de "AltGr" como en el teclado en 
español se puede usar alternativamente la tecla "*" o el teclado en pantalla
